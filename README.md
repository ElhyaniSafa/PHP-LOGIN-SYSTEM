
# 🔐 PHP Login System

A simple authentication system built using PHP and sessions.  
It allows users to register, login, and access a protected home page.

---

## 🚀 Features

- User registration
- User login
- Password hashing (secure)
- Session-based authentication
- Protected home page
- Simple responsive UI
